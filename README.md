# Transaction Management Selenium

Automation test scripts for Transaction Management Web Application using Selenium and Java.

## Project Structure

- `src/test/java/pages/` - Page Object Model for each UI page
- `src/test/java/tests/` - Test classes with scenarios

## How to Use
- Requires Java and Maven
- Selenium WebDriver dependency
- Clone, run `mvn test` in the project root

---

This project provides sample end-to-end test scripts covering main functional, negative, boundary, and exploratory test cases for transaction management as described in user stories.
